## Core thesis

> Из разговоров в состояние. Из состояния в действия.

Alaya — не просто memory layer для LLM. Это система, которая:
1. Принимает поток сырых событий (chats, calls, transcripts, integrations).
2. Извлекает сущности, факты, связи, обязательства.
3. Собирает рабочую модель мира (canonical state).
4. Запускает workflows по изменениям состояния (reminders, follow-ups, reviews).

Продуктовые векторы:
- **AlayaOS** — B2B AI chief of staff (текущий продукт)
- **Between** — personal / relationship copilot
- **Alaya Core** — гипотетическое общее ядро (пока не продукт)
